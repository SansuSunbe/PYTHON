# 08강 연산자 2(조건식)

# 비트

- 0또는 1
- 10진수 :
    - 0 ~ 9로 표현한 진법
- 2진수 :
    - 0 또는 1로 표현한 진법
- 1byte = 8bit

## 논리 연산자

| 기호 | 해석 | 의미 |
| --- | --- | --- |
| & | AND  | 두 비트가 모두 1이면 1 |
| | | OR  | 둘 중 하나라도 1이면 1 |
| ^ | XOR | 두 비트가 서로 다르면 1 |

## 단항 연산자

| 기호 | 해석 | 의미 |
| --- | --- | --- |
| ~ | NOT | 0을 1로, 1을 0으로 바꿔준다. |

## 부호 비트

- 최상위 비트를 부호비트라고 하며,
양수는 0, 음수는 1로 표현한다.
