# 09강 연산자 3(비트연산)

# 비트 연산 정리

## 논리 연산자

| 기호 | 의미 |
| --- | --- |
| & | 논리곱 |
| | | 논리합 |
| ^ | 베타 논리합 |

## 단항 연산자

| 기호 | 의미 |
| --- | --- |
| ~ | 논리 부정 |

## 쉬프트 연산자

- 비트가 넘어가면 잘리고, 빈 자리는 0으로 채워진다(음수는 빈 자리가 1로 채워진다)

| 기호 | 의미 |  |
| --- | --- | --- |
| >> | A >> B | A를 오른쪽으로 B만큼 비트 이동 |
| << | A << B | A를 왼쪽으로 B만큼 비트 이동 |
